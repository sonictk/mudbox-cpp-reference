53 /*
54  * The following macro invocations define the supported CG basic data types.
55  *
56  * The macros have the form :
57  *
58  * CG_DATATYPE_MACRO(name, compiler_name, enum_name, base_name, nrows, ncols, pc_name)
59  *
60  * name : The name of the data type.
61  * compiler_name : The name of the data type within the compiler syntax.
62  * enum_name : The C enumerant.
63  * base_name : The C enumerant of the base type.
64  * nrows : Number of rows for matrix types. Should be 0 other-wise.
65  * ncols : Number of columns for scalar, vector, and matrix types.
66  * pc_name : The C enumerant of the parameter class.
67  *
68  */
