55 /*
56  * The following macro invocations define the supported CG basic hardware
57  * bind locations.
58  *
59  * The macros have the form :
60  *
61  * CG_BINDLOCATION_MACRO(name, compiler_name, enum_int)
62  *
63  * name : The name of the location.
64  * enum_name : The C enumerant.
65  * compiler_name : The name of the location within the compiler syntax.
66  * int_id : Integer enumerant associated with this bind location.
67  * (3256 is reservered for CG_UNDEFINED)
68  * addressable : The bind location must have an integer address
69  * associated with it.
70  * ParamType : the cgParamType of this register.
71  *
72  */
